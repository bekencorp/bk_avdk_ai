使用说明：
1、解压bk_avi.rar压缩文件，解压后的文件结构不要改动；
2、将sample.avi替换为需要转换的avi文件；
3、打开cmd，在windows命令行执行bk_avi.exe文件，命令示例如下：
xxx\xxx\bk_avdk\components\multimedia\tools\aviconvert\bk_avi\bk_avi.exe -i xxx\xxx\bk_avdk\components\multimedia\tools\aviconvert\bk_avi\xxx.avi -f 25
-i：后面传入要转换的avi文件
-f：fps
4、处理完成后，会在同级目录下，生成一个avi文件。
